2、分别构造一个2×3的矩阵和2×2矩阵，将两个矩阵拼接成一个矩阵，写出命令和运行结果，出现的变量都用自己姓名的首字母开头。

---

代码：

```matlab
SY43_2x3 = [111,112,113;121,122,123]
SY43_2x2 = [211,212;221,222]
SY43_2x5 = [SY43_2x3,SY43_2x2]
```

结果：

```
>> SY43_2x3 = [111,112,113;121,122,123]
SY43_2x3 =
   111   112   113
   121   122   123
>> SY43_2x2 = [211,212;221,222]
SY43_2x2 =
   211   212
   221   222
>> SY43_2x5 = [SY43_2x3,SY43_2x2]
SY43_2x5 =
   111   112   113   211   212
   121   122   123   221   222
>> 
```
