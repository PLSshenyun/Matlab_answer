**sum的帮助命令：**

```matlab
help sum
```

**简述功能：**

sum - Sum of array elements
This MATLAB function returns the sum of the elements of A along the first array. dimension whose size does not equal 1.

数组元素总和
返回 A 沿大小不等于 1 的第一个数组维度的元素之和。

**使用方法：**


```matlab
S = sum(A)
S = sum(A,'all')
S = sum(A,dim)
S = sum(A,vecdim)
S = sum(___,outtype)
S = sum(___,nanflag)
```
